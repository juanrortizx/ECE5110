All latex files and pdf documents are in the second repository for this class.

Python:

Follow the differentiation, trapezoidal integration, and Simpson integration skills exactly, and generate all required outputs in `unit03/results/`.

Latex:

Write the article first, then generate the slides as a concise summary of that article.
