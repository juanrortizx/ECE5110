Given the testing data, we can now approximate a S-Curve that will pass through each of those points.

\vspace{0.2in}
\noindent \textbf{Lagrange Interpolation Method:}
\vspace{0.2in}

\noindent For each point ($x_k$, $y_k$), a "mini-curve" basis polynomial $L_k(x)$ is created, where it equals 1 at $x_k$ and 0 at all other data points. 
\vspace{0.2in}

\noindent The final S-Curve will be the SUM of all the mini-curves scaled by $y_k$:
\[\hspace{0.6in}
P(x) = y_0L_0(x) + y_1L_1(x) + ... + y_{n-1}L_{n-1}(x)
\]
\[\hspace{0.6in}
P(x) = \sum_{k=0}y_k\frac{\Pi(X-x_i)}{\Pi(x_k-x_i)}
\]

\leftskip 0in \rightskip 0in 

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\newpage % Page 5
\slideheader{Unit 02: Polynomial Interpolation}{ECE5110 - Numerical Modeling}

\leftskip 0.7in \rightskip 0.7in 
\vspace{0.4in}
\noindent In Python, we can compute the denominator by:
\vspace{0.2in}

\begin{center}
\begin{minipage}{0.75\linewidth} % adjust width as you like
\begin{minted}[fontsize=\huge, frame=single]{python}
diff = x[k] - np.delete(x, k)
denom[k] = np.prod(diff)
\end{minted}
\end{minipage}
\end{center}

\noindent \hspace{4.5in} equivalently: ${\Pi_{i \neq k}(x_k-x_i)}$
\vspace{0.2in}

\noindent Followed by building the numerator:

\begin{center}
\begin{minipage}{0.75\linewidth}
\begin{minted}[fontsize=\huge, frame=single]{python}
num = np.ones_like(X)
for i in range(n):
    if i == k: continue
    num *= (X - x[i])
\end{minted}
\end{minipage}
\end{center}

\noindent \hspace{4.5in} equivalently: ${\Pi_{i \neq k}(X - x_i)}$

\leftskip 0in \rightskip 0in

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

\newpage % Page 6
\slideheader{Unit 02: Polynomial Interpolation}{ECE5110 - Numerical Modeling}

\leftskip 0.7in \rightskip 0.7in 
\vspace{0.4in}
\noindent Lastly, the code will build the interpolation:
\vspace{0.2in}

\begin{center}
\begin{minipage}{0.75\linewidth} % adjust width as you like
\begin{minted}[fontsize=\huge, frame=single]{python}
out += y[k] * (num / denom[k])

\end{minted}
\end{minipage}
\end{center}

\noindent \hspace{4.5in} equivalently: $\sum_{k=0}y_k\frac{\Pi(X-x_i)}{\Pi(x_k-x_i)}$
\vspace{0.2in}

\noindent This ensures the output matches the exact data point and fills in the curve smoothly for every value in between.

\begin{itemize}
    \item The plot represents the static plant model (input $\rightarrow$ steady-state output) of the brushed DC motor.
    \item A deadzone appears at low voltages due to mechanical friction.
    \item The motor speed increases rapidly through the mid-voltage region.
    \item At higher voltages, the motor approaches saturation, where additional voltage results in only small increases in RPM.
\end{itemize}

\vspace{0.2in}
Based on these findings, we can tune a PID (Proportional Derivative Integral) controller to the characteristics of the specific brushed DC motor.
