\noindent Given the function $f(x) = x^2$, we will explore how to compute both it's derivative and integral using numerical methods in Python.

\vspace{0.2in}
\noindent {\LARGE \textbf{Derivative Approximation:}}
\vspace{0.1in}
{\Large
\begin{itemize}[leftmargin=2.5in]
    \item Forward Difference: $f'(x) = \frac{x_{n+1} - f(x_n)}{x_{n+1} - x_n} + O(x_{n+1} - x_n)$
    \item Backward Difference: $f'(x) = \frac{x_{n-1} - f(x_n)}{x_{n-1} - x_n} + O(x_{n-1} - x_n)$
    \item Central Difference: $f'(x) = \frac{x_{n-1} - f(x_n)}{x_{n-1} - x_n} + O(x_{n+1})^2$
\end{itemize}
}
\vspace{0.2in}
\noindent {\LARGE \textbf{Integral Approximation:}}
\vspace{0.1in}
{\Large
\begin{itemize}[leftmargin=2.5in]
    \item Trapezoidal Rule: $\int_a^b f(x) dx = h(\frac{f(a)+f(b)}{2}) + O(h^3)$
    \item Simpson's Rule: $\int_{x_0}^{x_2} f(x) dx = h\frac{f(x_0) + 4f(x_1) + f(x_2)}{3} + O(h^5)$
    \item Midpoint Rule: $\int_a^b f(x) dx = h f(\frac{a+b}{2}) + O(h^3)$
\end{itemize}
}

The derivative of $f(x) = x^2$ is $f'(x) = 2x$. \\We can approximate this derivative at x = 1, where $f'(1) = 2$.

\vspace{0.2in}
\noindent {\LARGE \textbf{Python Implementation:}}
\begin{center}
\begin{minipage}{0.75\linewidth}
\begin{minted}[fontsize=\large, frame=single]{python}

    def numerical_differentiation_3point(self,
                                        f, 
                                        x, 
                                        h=1e-5, 
                                        method='central'
                                        ):
        if method == 'central':
            return (f(x + h) - f(x - h)) / (2 * h)
        elif method == 'forward':
            return (-3 * f(x) + 4 * f(x + h) - f(x + 2 * h)) / (2 * h)
        elif method == 'backward':
            return (f(x - 2 * h) - 4 * f(x - h) + 3 * f(x)) / (2 * h)
        else:
            raise ValueError("must be 'central', 'forward', or 'backward'")

\end{minted}
\end{minipage}
\end{center}

We can see that the central difference method provides the lowest error, which is consistent with its second order accuracy. 

\vspace{0.2in}
\noindent The error term is proportional to $h^2$, which shrinks much faster as h gets smaller.

\vspace{0.2in}
\noindent Forward and backward use $f(x+h)$ and $f(x-h)$ respectively, while the central difference uses both:

\begin{center}
$\frac{f(x+h)-f(x-h)}{2h}$
\end{center}

Now the integral of $f(x) = x^2$ from 0 to 1 is $\int_0^1 x^2 dx = \frac{1}{3}$.

\noindent {\Large \textbf{Midpoint Rule:}}
\begin{center}
\begin{minipage}{0.75\linewidth}
\begin{minted}[fontsize=\large, frame=single]{python}
    def midpoint_rule(self, f, a, b, n):
        if n <= 0:
            raise ValueError("Num of sub-ints n must be positive.")
        
        a = float(a); b = float(b)
        h = (b - a) / n # width of each subinterval
        midpoints = a + h * (np.arange(n) + 0.5) # midpoints of subintervals

        return h * np.sum(f(midpoints)) # sum of f at midpoints times width
\end{minted}
\end{minipage}
\end{center}

\noindent {\Large \textbf{Trapezoidal Rule:}}
\begin{center}
\begin{minipage}{0.75\linewidth}
\begin{minted}[fontsize=\large, frame=single]{python}
    def trapezoidal_rule(self, f, a, b, n):
        if n <= 0:
            raise ValueError("Num of sub-ints n must be positive.")
        
        a = float(a); b = float(b)
        h = (b - a) / n # width of each subinterval
        x = a + h * np.arange(n + 1) # endpoints of subintervals

        return (h / 2) * (f(x[0]) + 2 * np.sum(f(x[1:n])) + f(x[n])) # tz sum
\end{minted}
\end{minipage}
\end{center}

\noindent {\Large \textbf{Simpson's Rule:}}
\begin{center}
\begin{minipage}{0.75\linewidth}
\begin{minted}[fontsize=\large, frame=single]{python}
    def simpsons_rule(self, f, a, b, n):
        if n <= 0 or n % 2 != 0:
            raise ValueError("Num of sub-ints n must be a posi-even int.")
        
        a = float(a); b = float(b)
        h = (b - a) / n # width of each subinterval
        x = a + h * np.arange(n + 1) # endpoints of subintervals

        return (h / 3) * (f(x[0]) + 
                          4 * np.sum(f(x[1:n:2])) + 
                          2 * np.sum(f(x[2:n-1:2])) + 
                          f(x[n])) # Simpson's sum
\end{minted}
\end{minipage}
\end{center}

Based on the error values, we can see that Simpson's rule provides a significantly more accurate approximation. The key differences are the order of accuracy:

\vspace{0.2in}
\noindent {\LARGE \textbf{Midpoint Rule:}}
{\Large
\begin{itemize}[leftmargin=3in]
    \item Error is proportional to $h^3$
\end{itemize}
}
\vspace{0.2in}
\noindent {\LARGE \textbf{Trapezoidal Rule:}}
{\Large
\begin{itemize}[leftmargin=3in]
    \item Error is proportional to $h^3$
\end{itemize}
}
\vspace{0.2in}
\noindent {\LARGE \textbf{Simpson's Rule:}}
{\Large
\begin{itemize}[leftmargin=3in]
    \item Error is proportional to $h^5$
    \item Fifth-order accuracy means error decreases much faster as h gets smaller compared to the other two methods. \rightskip 3.5in
\end{itemize}
}
