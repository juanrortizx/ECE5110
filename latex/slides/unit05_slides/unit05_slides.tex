{\LARGE Modeling and Simulation of a Temperature Sensor Using the Runge–Kutta Method}

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\newpage %Page 2
\slideheader{Unit 05: Ordinary Differential Equations}{ECE5110 - Numerical Modeling}

\leftskip 0.7in \rightskip 0.7in 
\vspace{0.4in}
\noindent Despite temperature sensors being in controlled environments, they can still be affected by external noise, such electromagnetic interference or mechanical vibrations. 
\vspace{0.1in}

We will model a noisy temperature sensor's output response to a step change in temperature using an ordinary differential equation (ODE).
\vspace{0.1in}

\noindent First order ODE form:

\[\hspace{0.4in}
\frac{dT}{dt} = -a(T - T_{ambient}) + K_{p}(T_{setpoint} - T)
\]

\begin{center}
\begin{minipage}{0.75\linewidth} % adjust width as you like
\begin{minted}[fontsize=\Large, frame=single]{python}
def f(t, y):
    return -a * (y - T_ambient) 
                  + Kp * (T_set - y)
\end{minted}
\end{minipage}
\end{center}


\vspace{0.2in}

\leftskip 0in \rightskip 0in 

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\newpage %Page 3
\slideheader{Unit 05: Ordinary Differential Equations}{ECE5110 - Numerical Modeling}

\leftskip 0.7in \rightskip 0.7in 
\vspace{0.4in}
\noindent Each term in the ODE has a specific physical meaning:
\vspace{0.4in}

\begin{minipage}[t]{\dimexpr\textwidth-1.2in}
\begin{itemize}
    \item $T$: sensor temperature
    \item $t$: time
    \item $a$: positive cooling rate constant
    \item $T_{ambient}$: ambient temperature
    \item $K_{p}$: proportional gain constant
    \item $T_{setpoint}$: desired temperature
\end{itemize}
\end{minipage}


\vspace{0.2in}

\leftskip 0in \rightskip 0in 

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\newpage %Page 4
\slideheader{Unit 05: Ordinary Differential Equations}{ECE5110 - Numerical Modeling}

\leftskip 0.7in \rightskip 0.7in 
\vspace{0.4in}
\noindent We can solve this ODE by implementing the 4th order Runge-Kutta method.
\vspace{0.4in}

\noindent RK4 advances the solution using: 
\begin{align*}
k_1 &= f(t_n, y_n) \\
k_2 &= f(t_n + \frac{h}{2}, y_n + \frac{h}{2}k_1) \\
k_3 &= f(t_n + \frac{h}{2}, y_n + \frac{h}{2}k_2) \\
k_4 &= f(t_n + h, y_n + hk_3) \\
y_{n+1} &= y_n + \frac{h}{6}(k_1 + 2k_2 + 2k_3 + k_4)
\end{align*}




\leftskip 0in \rightskip 0in 

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

\newpage %Page 5
\slideheader{Unit 05: Ordinary Differential Equations}{ECE5110 - Numerical Modeling}

\leftskip 0.7in \rightskip 0.7in 
\vspace{0.4in}

\noindent This can be implemented in Python as follows:: 
\begin{center}
\begin{minipage}{0.75\linewidth} % adjust width as you like
\begin{minted}[fontsize=\huge, frame=single]{python}
for i in range(1, len(steps)):
  x_prev = steps[i - 1]
  y_prev = y[i - 1]
  k1 = f(x_prev, y_prev)
  k2 = f(x_prev + h / 2, 
         y_prev + h * k1 / 2)
  k3 = f(x_prev + h / 2, 
         y_prev + h * k2 / 2)
  k4 = f(x_prev + h, 
         y_prev + h * k3)
  y[i] = y_prev + h * (k1 + 2 * k2 + 
                            2 * k3 + 
                                k4) / 6
\end{minted}
\end{minipage}
\end{center}

\leftskip 0in \rightskip 0in

\noindent To simulate the sensor noise, we add a random noise term to the ODE output at each time step:

\begin{center}
\begin{minipage}{0.75\linewidth} % adjust width as you like
\begin{minted}[fontsize=\huge, frame=single]{python}
noise = np.random.normal(0.0, 
                         noise_std, 
                         size=len(y)
                         )
y_noisy = y + noise
\end{minted}
\end{minipage}
\end{center}
