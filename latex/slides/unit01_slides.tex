\vspace{0.4in}
You have been hired to help design the input stage of a new guitar effects pedal.
This pedal needs a low‑pass RC filter to remove harsh high‑frequencies while keeping the musical tone warm and natural.

\vspace{0.4in}
\noindent
The design team wants the filter’s cutoff frequency to be:
\vspace{0.1in}
\begin{center}$f_c = 1kHz$\end{center}
\vspace{0.1in}
The resistor value is already chosen:
\begin{center}$R = 10k\Omega$\end{center}
\vspace{0.1in}
Your job is to determine the capacitor value C that will produce the desired cutoff

\leftskip 0in \rightskip 0in 

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\newpage %Page 3
\slideheader{Newton Example: Designing a tone-shaping circuit for a guitar pedal}{Newton Method}

\leftskip 0.7in \rightskip 0.7in 
\vspace{0.4in}
\noindent
The cutoff frequency of a simple RC low‑pass filter is:
\begin{center}$f_c = \frac{1}{2\pi RC}$\end{center}
\vspace{0.2in}
This can be solved by rearranging the equation in root-finding form:
\begin{center}$g(C) = \frac{1}{2\pi RC} - f_c = 0$\end{center}
\vspace{0.2in}
Next define the function and it's derivative:
\vspace{0.2in}
\begin{center}$f(C) = \frac{1}{2\pi RC} - f_c$
\vspace{0.2in}
\\$f'(C) = -\frac{1}{2\pi RC^2}$\end{center}

\leftskip 0in \rightskip 0in

Given the 1nF–100nF capacitance range used in guitar‑pedal signal paths, we can make a few reasonable assumptions when applying Newton’s Method to solve our filter equation.
\vspace{0.2in}
{
\begin{itemize}
    \item The plot interval: \{START, STOP\} = \{1nF, 100nF\} = \{1e-9, 1e-7\}
    \item Accuracy and tolerance: PRECISION = 1nF = 1e-9
    \item Initial guess: $x_0$ = 10nF = 1e-8
    \item Iterations: MAX\_STEPS = 20 (NM converges fast)
\end{itemize}
}
\vspace{0.2in} 
Recall the defined functions from the previous slides:
\vspace{0.2in}
{
\begin{itemize}
    \item f = 1/(2*math.pi*10000*C) - 1000
    \item df = -1/(2*math.pi*10000*C**2)
\end{itemize}
}

\begin{minted}[fontsize=\LARGE]{python}
def solve_newton_method(self, f, df, x0, tolerance, max_steps):
    x = x0

    for i in range(1, max_steps + 1):
        dfx = df(x)
        if dfx == 0:
            return x, i, -1

        fx = f(x)
        x_new = x - fx / dfx

        if abs(x_new - x) < tolerance:
            return x_new, i, 0

        x = x_new

    return x, max_steps, -2
\end{minted}
